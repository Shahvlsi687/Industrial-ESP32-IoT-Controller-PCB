# Industrial-ESP32-IoT-Controller-PCB
Designed and routed a manufacturable IoT board in KiCad pairing an ESP32-WROOM-32 with an I2C OLED display interface and an onboard 3.3V voltage regulator network. Optimized RF performance using an antenna Keep-Out zone and resolved power spikes with localized SMD decoupling. Generated production-ready Gerber plots with zero DRC errors.
